  2. MergePresetFiles("myapi") ‚Üí merges request.toml + headers.toml + body.toml + query.toml

  4. PromptForVariables() ‚Üí prompts user:

  7. BuildHTTPRequest() ‚Üí creates go-resty request with all components
  8. ExecuteHTTPRequest() ‚Üí sends HTTP request
  9. DisplayResponse() ‚Üí shows formatted response with status, headers, pretty JSON

‚Üí Same flow but prompts for hard variables too and saves new values to variables.toml
